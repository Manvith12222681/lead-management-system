# Lead Management System

## Overview
A backend-driven Lead Management System developed from scratch to manage, score, and track leads efficiently.  
The system automates lead follow-ups based on lead scoring logic, helping businesses improve customer engagement and sales workflows.

## Features
- Lead creation and management
- Automated lead scoring logic
- Hierarchical lead tracking
- Backend-driven workflow automation
- MongoDB-based data persistence

## Tech Stack
- Node.js
- MongoDB (Native Driver)
- JavaScript
- HTML & CSS

## System Design
- Backend-first architecture
- Direct MongoDB integration without frameworks
- Modular code structure for maintainability
- Designed with scalability in mind

## How to Run
1. Clone the repository  
2. Install dependencies  
   `npm install`
3. Configure MongoDB connection
4. Run the application  
   `node index.js`
